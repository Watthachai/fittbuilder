05 · ฝั่ง CRN

CRN ทำอะไรกับ payload

handleIngest — ลำดับจริงตอนรับ body

1

Ensure org — parse org_id (ว่าง → default), upsert org ด้วย org_name (ว่าง → "FBD Default")

2

Ensure project — parse project_id; ยังไม่มีในระบบ → สร้างใหม่ (re-export ใช้ id เดิมซ้ำได้)

3

Enqueue build — re-marshal ingestBody ทั้งก้อน (รวม zip_base64) เป็น job payload → jobs layer materialize ไฟล์ + ประกอบ Claude prompt จาก idea/brd/prd/prompts → ตอบ 202
